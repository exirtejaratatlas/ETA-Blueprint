# ETA Iconography System

## Philosophy

ETA icons represent engineering precision, enterprise clarity, and modern digital systems.

Icons should simplify information, not decorate interfaces.

The icon language must remain consistent across every ETA product.

---

# Visual Style

Style:

- Outline
- Minimal
- Geometric
- Professional
- Technical

Avoid:

- Filled icons
- Cartoon style
- Decorative graphics
- 3D icons
- Emoji-style icons

---

# Corner Radius

Use soft rounded corners where appropriate.

Sharp industrial angles should be avoided unless representing mechanical components.

# Accessibility

Every icon must have:

- Accessible label
- Consistent meaning
- Sufficient contrast
- Clear visual recognition

Icons must never be the sole indicator of status.

Always pair icons with text where required.
