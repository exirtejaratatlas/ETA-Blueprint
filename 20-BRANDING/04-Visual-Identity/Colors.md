# ETA Color System

## Philosophy

ETA's color system reflects engineering precision, industrial strength, enterprise reliability, and modern artificial intelligence.

The palette is intentionally minimal to ensure timelessness and consistency across all digital and printed materials.

# Usage Rules

Primary color dominance:

70%

Neutral colors:

20%

Accent color:

10%

Copper should only be used to draw attention to important actions, highlights, and premium elements.

---

# Accessibility

All color combinations must comply with WCAG AA contrast requirements.

Color should never be the only indicator of meaning; icons, labels, and typography must reinforce status.

---

# Visual Impression

The ETA palette should communicate:

- Trust
- Stability
- Intelligence
- Industrial Expertise
- Premium Quality
- Modern Technology

# ETA Typography System

## Philosophy

Typography is one of the strongest elements of the ETA identity.

It should communicate precision, professionalism, clarity, and modern enterprise technology.

The typography system must remain clean, highly readable, and consistent across all platforms.

---

# Primary Typeface

## Inter

Usage:

- Website
- Dashboard
- Enterprise Platform
- UI Components
- Presentations

Characteristics:

- Modern
- Highly readable
- Excellent digital rendering
- Enterprise appearance

---

# Persian Typeface

## Vazirmatn

Usage:

- Persian Website
- Dashboards
- Mobile Applications
- Company Documents

Characteristics:

- Modern Persian typography
- Excellent readability
- UI optimized
- Open Source

---

# Monospace Typeface

---

# Usage Rules

- Never use more than two font families.
- Maintain a clear visual hierarchy.
- Avoid excessive bold text.
- Use whitespace to improve readability.
- Keep typography consistent across all ETA products.

---

# Design Goal

ETA typography should feel:

- Professional
- Calm
- Technical
- Premium
- Enterprise Ready
- Easy to Read

Typography must support information—not compete with it.
