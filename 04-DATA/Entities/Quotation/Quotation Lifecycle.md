# Quotation Lifecycle

## Purpose

The Quotation Lifecycle defines the complete journey of a supplier quotation from creation until procurement award and archival.

The lifecycle ensures engineering validation, commercial transparency, supplier fairness, procurement governance, and AI-assisted decision making.

---

```

Submitted

↓

↓
```

---

# Stage 1 — Draft

Supplier prepares the quotation.

Activities

- Commercial proposal
- Technical proposal
- Documents
- Pricing

Status

```
Draft
```

---

# Stage 2 — Submitted

Quotation is officially submitted.

Required

- Commercial Offer
- Technical Offer
- Attachments

Status

```
Submitted
```

---

# Stage 3 — Technical Review

Engineering evaluates:

- Technical compliance
- Datasheets
- Drawings
- Material
- Standards
- Deviations

Status

```
Technical Review
```

---

# Stage 4 — Commercial Review

Procurement evaluates:

- Price
- Delivery
- Payment Terms
- Warranty
- Incoterms
- Commercial Risks

Status

```
Commercial Review
```

---

# Stage 5 — Clarification

Supplier provides:

- Missing documents
- Revised technical proposal
- Commercial clarification

Status

```
Clarification
```

---

# Stage 6 — Negotiation

Commercial negotiation may include:

- Price reduction
- Delivery optimization
- Warranty
- Payment terms
- Commercial exceptions

Every negotiation creates a new quotation revision.

Status

```
Negotiation
```

---

# Stage 7 — Final Evaluation

ETA combines:

- Technical Score
- Commercial Score
- Supplier Performance
- AI Recommendation
- Risk Assessment

Status

```
Final Evaluation
```

---

# Stage 8 — Award Decision

Procurement selects:

- Winning Supplier
- Split Award
- No Award

Executive approval may be required.

Status

```
Awarded
```

---

# Stage 9 — Purchase Order

Awarded quotation generates:

- Purchase Order
- Contract reference
- Procurement commitments

Status

```
PO Created
```

---

# Stage 10 — Closed

Quotation is finalized.

No further revisions are allowed.

Status

```
Closed
```

---

# Stage 11 — Archived

Historical quotations remain available for:

- Audit
- Analytics
- Supplier Performance
- AI Learning
- Procurement Benchmarking

Status

```
Archived
```
