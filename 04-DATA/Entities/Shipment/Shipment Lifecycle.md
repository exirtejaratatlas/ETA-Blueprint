# Shipment Lifecycle

## Purpose

The Shipment Lifecycle defines the complete operational journey of a shipment from logistics planning through transportation, customs clearance, warehouse receiving, inventory synchronization, and archival.

The lifecycle ensures complete visibility, operational control, financial synchronization, procurement governance, and AI-assisted logistics monitoring.

- Select Supplier
- Select Carrier
- Define Shipment Type
- Define Transport Mode
