# Invoice Lifecycle

## Purpose

The Invoice Lifecycle defines the complete journey of an invoice from creation through validation, approval, accounting, payment, reconciliation, and archival.

The lifecycle ensures complete financial governance, procurement synchronization, tax compliance, accounting integrity, payment control, AI-assisted financial intelligence, and full auditability.

---

# Stage 1 — Draft

Invoice is created.

Activities

- Create invoice
- Add invoice lines
- Select supplier/customer
- Calculate taxes

Status

```
Draft
```

---

# Stage 2 — Submitted

Invoice is submitted for validation.

Activities

- Submit invoice
- Lock financial values
- Notify reviewers

Status

```
Submitted
```

---

# Stage 3 — Three-Way Matching

System validates invoice against procurement records.

Activities

- Match Purchase Order
- Match Goods Receipt
- Match Contract
- Detect discrepancies

Status

```
Matching
```

---

# Stage 4 — Under Review

Finance reviews the invoice.

Activities

- Financial validation
- Tax verification
- Cost verification
- Exception handling

Status

```
Under Review
```

---

# Stage 5 — Approved

Invoice receives formal approval.

Activities

- Manager approval
- Financial approval
- Procurement approval (if required)

Status

```
Approved
```

---

# Stage 6 — Posted

Invoice is posted to accounting.

Activities

- Create journal entries
- Update accounting
- Update tax ledger
- ERP synchronization

Status

```
Posted
```

---

# Stage 7 — Partially Paid

Invoice has received one or more payments.

Activities

- Register payment
- Update outstanding balance
- Recalculate payable amount

Status

```
Partially Paid
```

---

# Stage 8 — Paid

Invoice has been fully settled.

Activities

- Complete payment
- Close payable
- Confirm settlement

Status

```
Paid
```

---

# Stage 9 — Reconciled

Financial reconciliation is completed.

Activities

- Bank reconciliation
- Ledger reconciliation
- Payment verification

Status

```
Reconciled
```

---

# Stage 10 — Closed

Invoice processing is finalized.

Activities

- Final financial validation
- Audit verification
- KPI calculation

Status

```
Closed
```

---

# Stage 11 — Archived

Historical invoice remains available.

Activities

- Financial audit
- Compliance review
- Tax reporting
- AI learning
- Executive reporting

Status

```
Archived
```
