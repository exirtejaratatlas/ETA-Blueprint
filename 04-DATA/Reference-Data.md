# ETA Reference Data

## Purpose

Reference Data contains standardized values shared across the ETA Enterprise Procurement Ecosystem.

Unlike Master Data and Transaction Data, Reference Data changes infrequently and provides consistent lookup values used by ERP, CRM, AI, APIs, analytics, and engineering systems.

- Single Enterprise Standard
