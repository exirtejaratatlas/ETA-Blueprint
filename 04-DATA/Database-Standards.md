# ETA Database Standards

## Purpose

This document defines the enterprise database standards for the ETA Enterprise Procurement Ecosystem.

These standards ensure consistency, maintainability, scalability, performance, and security across all databases used by Exir Tejarat Atlas.

---

# Database Platform

Primary Database

PostgreSQL 17+

Character Encoding

UTF-8

Timezone

UTC

Primary Key Strategy

UUID v7 (preferred)

Fallback

UUID v4
